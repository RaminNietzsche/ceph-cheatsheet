# توسعه

این بخش قرار است **راهنمای توسعه‌دهنده Ceph** باشد: راه‌اندازی build، قراردادهای کدنویسی، تست و گردش‌کار مشارکت.

## وضعیت

**به‌زودی** — محتوا هنوز نوشته نشده است.

## موضوعات برنامه‌ریزی‌شده

- ساخت Ceph از source
- گردش‌کار توسعه RGW
- اجرای تست و linter
- ارسال patch به upstream
