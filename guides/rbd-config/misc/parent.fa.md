### rbd_parent_cache_enabled

| | |
|---|---|
| نوع | Bool · default `False` · **Advanced** |
| جدول | [rbd.md#SP_rbd_parent_cache_enabled](../../../config/rbd/rbd.md#SP_rbd_parent_cache_enabled) |

**کارکرد:** whether to enable rbd shared ro cache

**زمان استفاده:** به‌طور پیش‌فرض غیرفعال است؛ وقتی به این قابلیت نیاز دارید و مبادله‌های آن را می‌پذیرید، فعال کنید.

**مثال:**

```bash
ceph config set client rbd_parent_cache_enabled true
ceph config get client rbd_parent_cache_enabled
```

**یافتن مقدار بهینه:**

**مدل تنظیم:** Policy

1. مستند کنید چرا `False` برای سیاست شما درست است.
2. فقط برای الزامات سازگاری یا امنیت تغییر دهید.
3. پس از تغییرات، روندهای کاری کلاینت و مدیر را آزمایش کنید.
**شاخص‌های پایش:** `ceph -s`، slow ops، شمارنده‌های عملکرد دیمن، صف بازیابی و scrub.

```bash
ceph config get client rbd_parent_cache_enabled
ceph -s
# گزینه‌های کلاینت: در بخش client یا ceph.conf تنظیم کنید
```
