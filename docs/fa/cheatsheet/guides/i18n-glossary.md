# واژه‌نامهٔ اصطلاحات Ceph

برای یکنواختی در صفحات فارسی از این معادل‌ها استفاده کنید. **دستورات CLI، کلید config و نام دیمن را ترجمه نکنید.**

## دیمن‌ها و اجزا

| انگلیسی | فارسی | یادداشت |
|---------|-------|---------|
| Monitor (MON) | مانیتور | MON |
| OSD | دیمن ذخیره‌سازی شیء (OSD) | |
| MGR | Manager | نام ماژول انگلیسی |
| MDS | سرور متادیتا (MDS) | |
| RGW | دروازه RADOS (RGW) | |
| PG | گروه placement (PG) | |
| CRUSH | CRUSH | نام الگوریتم |

## RGW

| انگلیسی | فارسی |
|---------|-------|
| Bucket | bucket |
| Quota | سهمیه (quota) |
| Realm / zone / zonegroup | realm / zone / zonegroup |
| Multisite | چندسایته |
| SAL | لایه انتزاع store (SAL) |

## قوانین ترجمه

1. **متن** — عنوان، برچسب جدول و توضیح را ترجمه کنید.
2. **کد** — دستور، flag و نام گزینهٔ config انگلیسی بماند.
3. **Placeholder** — `<user_id>`، `<bucket_name>`؛ [راهنمای سبک CLI](cli-style-guide.md).
