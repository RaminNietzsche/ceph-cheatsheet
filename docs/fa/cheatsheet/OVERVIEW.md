# مرجع Ceph

مرجع آفلاین و ساده برای مدیران Ceph — بر اساس **نقش**، **مقیاس**، **خط فرمان (CLI)** و **پیکربندی**.

**منبع:** [ceph/ceph](https://github.com/ceph/ceph) شاخهٔ `main` — تاریخ تولید در [VERSION](../version.md).
