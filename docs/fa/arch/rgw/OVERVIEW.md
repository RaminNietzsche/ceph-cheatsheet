# Ceph RADOS Gateway — مستندات فنی

این سایت مستندات **توسعه‌دهندگان** برای دروازه شیء Ceph (RGW) است. هدف آن توضیح معماری، جریان درخواست، لایه‌های ذخیره‌سازی و مرجع نمادهای کد **بدون نیاز به مرور مستقیم کل مخزن** است.

## محدوده

| شامل | خارج از محدوده |
|------|----------------|
| معماری و توپولوژی زمان اجرا | گردش‌کار Git و استراتژی شاخه |
| ماژول‌های `src/rgw/` | خطوط CI/CD و استقرار خودکار |
| قطعه‌های زنده از کد منبع | تغییر رفتار سرویس در این مخزن مستندات |

## ساختار

- **برنامه یادگیری** — مسیر گام‌به‌گام درک عمیق کد ([شروع کن](learning-program/index.md))
- **معماری** — نمای کلی، توالی‌ها، محدودیت‌های HA
- **ماژول‌ها** — راهنمای بسته‌های اصلی با `--8<--` snippet
- **راهنماها** — استقرار، قرارداد مستندسازی، ارجاع به کد
- **پیوست** — فهرست نمادها و گراف include (با `make generate`)

## ساخت محلی

```bash
cd src/rgw/docs-extended
make install
make serve    # http://127.0.0.1:8000
```

## مخزن منبع

کد: [`src/rgw/`](https://github.com/ceph/ceph/tree/main/src/rgw) در پروژه [Ceph](https://github.com/ceph/ceph).
