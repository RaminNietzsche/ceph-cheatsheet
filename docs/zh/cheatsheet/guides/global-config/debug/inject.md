### inject_early_sigterm

| | |
|---|---|
| 类型 | Bool · default `False` · **Dev** |
| 表格 | [inject.md#SP_inject_early_sigterm](../../../config/global/inject.md#SP_inject_early_sigterm) |

**作用：** send ourselves a SIGTERM early during startup

**何时使用：** 仅用于开发、测试或 upstream 调试 — 不可用于生产调优。

**示例：**

```bash
ceph config set global inject_early_sigterm true
ceph config get global inject_early_sigterm
```

**寻找最优值：**

**调优模型：** 开发

1. 生产环境保持 upstream 默认值（`False`）。
2. 仅在实验室复现特定问题时修改。
3. 节点回到生产池前请还原。
